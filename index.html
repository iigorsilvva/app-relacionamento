inicial

Para ativar o feed social, notificações e o painel admin, você precisa conectar ao Firebase (gratuito).

Passo 1
Crie seu projeto Firebase
Acesse console.firebase.google.com, crie um projeto, ative o Firestore Database e o Authentication (método anônimo).
Passo 2
Copie a configuração do app
Em Configurações do Projeto → Seus apps → Web, copie o objeto firebaseConfig e cole abaixo:
Passo 3
Configure as Regras do Firestore
No Firestore → Regras, cole:

rules_version = '2';
service cloud.firestore {
 match /databases/{db}/documents {
  match /{doc=**} {
   allow read, write: if true;
  }
 }
}
